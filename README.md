## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/Alkasingh04/Python_Prgms1_InterviewQs.git
```

2. Navigate to the folder:

```
cd Python_Prgms1_InterviewQs
```

3. Run any program:

```
python filename.py
```
